# Script Vidéo - SmartQueue Mode d'Emploi

## Informations techniques

- **Durée totale** : 4 minutes 30 secondes
- **Format** : Screencast + texte à l'écran
- **Résolution** : 1920x1080 (Full HD)
- **Outil recommandé** : Loom, OBS Studio ou ScreenFlow
- **Voix** : Voix-off ou sous-titres

## Conseils de tournage

### Préparation
- **Nettoyer la base de données** avant l'enregistrement (données propres)
- **Préparer les emails** des comptes OWNER et STAFF à l'avance
- **Tester chaque étape** avant d'enregistrer pour éviter les erreurs
- **Thème clair** recommandé pour une meilleure visibilité en vidéo
- **Fermer les notifications** et onglets inutiles du navigateur

### Enregistrement
- **Résolution** : 1920x1080 (Full HD)
- **Navigateur** : Chrome ou Firefox en plein écran
- **Zoom** : 110-125% pour que le texte soit bien lisible
- **Curseur** : Utiliser un highlighter de curseur pour guider l'oeil
- **Pauses** : Marquer une pause de 1-2 sec entre chaque action importante

### Post-production
- **Couper** les temps de chargement trop longs
- **Ajouter** des annotations/flèches pour guider l'attention
- **Ajouter** les textes à l'écran (titres des étapes)
- **Musique** : Fond musical calme et professionnel (optionnel)
- **Sous-titres** : Fortement recommandés pour l'accessibilité

### Outils recommandés
- **Enregistrement** : Loom (simple), OBS Studio (gratuit, pro), ScreenFlow (Mac)
- **Montage** : DaVinci Resolve (gratuit), CapCut (simple), Adobe Premiere (pro)
- **Voix-off** : Audacity (gratuit), ElevenLabs (voix IA), micro USB
- **Annotations** : Canva (gratuit), After Effects (pro)
- **Musique libre** : Pixabay Music, YouTube Audio Library

## Remotion - Vidéo programmée en React

Le projet inclut un dossier `remotion-video/` avec 9 scènes animées en React.

### Structure du projet Remotion

```
remotion-video/
├── index.ts                    # Point d'entrée Remotion
├── Root.tsx                    # Composition principale (toutes les scènes)
├── components/
│   ├── AnimatedText.tsx        # Texte animé avec spring
│   ├── FadeTransition.tsx      # Fade in/out automatique
│   ├── MockBrowser.tsx         # Faux navigateur avec barre d'URL
│   ├── MockNavbar.tsx          # Fausse navbar SmartQueue animée
│   └── SceneTitle.tsx          # Badge "Étape X" + titre
├── compositions/
│   ├── Intro.tsx               # Scène 1 : Logo + titre animé
│   ├── Signup.tsx              # Scène 2 : Formulaire d'inscription
│   ├── CreateServices.tsx      # Scène 3 : Création des services
│   ├── CreatePosts.tsx         # Scène 4 : Création des postes
│   ├── AddStaff.tsx            # Scène 5 : Ajout et assignation du personnel
│   ├── PublicPage.tsx          # Scène 6 : Configuration page publique
│   ├── ClientFlow.tsx          # Scène 7 : Parcours client (ticket)
│   ├── StaffFlow.tsx           # Scène 8 : Parcours employé (traitement)
│   ├── Outro.tsx               # Scène 9 : Récapitulatif + CTA
│   └── FullVideo.tsx           # Enchaînement de toutes les scènes
└── assets/
    └── voiceover/              # Fichiers audio voix-off (à ajouter)
```

### Commandes

```bash
# Ouvrir Remotion Studio (preview en temps réel)
npm run video:preview

# Rendre la vidéo complète en MP4
npm run video:render

# Rendre une scène individuelle
npm run video:render-scene -- Scene1-Intro out/intro.mp4
```

### Compositions disponibles

| ID | Scène | Durée |
|----|-------|-------|

### Intégrer la voix-off

1. Générez les fichiers audio (voir section ElevenLabs ci-dessous)
2. Placez-les dans `remotion-video/assets/voiceover/`
3. Importez dans chaque scène :

```tsx
import { Audio } from "remotion";

// Dans le composant de la scène :
<Audio src={staticFile("voiceover/scene1.mp3")} />
```

### Configuration recommandée

- **Voix** : Adam (masculine, professionnelle) ou Rachel (féminine, chaleureuse)
- **Langue** : Français
- **Stabilité** : 65%
- **Clarté** : 80%
- **Style** : 30% (pour un ton naturel sans être trop expressif)
