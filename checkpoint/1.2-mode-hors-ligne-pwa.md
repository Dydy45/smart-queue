## 📋 Vue d'ensemble

Transformer SmartQueue en Progressive Web App (PWA) pour permettre aux utilisateurs d'installer l'application sur leur appareil mobile/desktop et de continuer à l'utiliser même sans connexion internet. Cela améliore considérablement l'expérience utilisateur, notamment dans les zones avec une connexion instable.

### Bénéfices attendus
- ✅ **Installation native** : L'app peut être installée sur mobile/desktop comme une app native
- ✅ **Fonctionnement hors-ligne** : Consultation des tickets en cache même sans internet
- ✅ **Performance améliorée** : Chargement instantané grâce au cache
- ✅ **Expérience utilisateur optimale** : Pas de barre d'adresse, plein écran
- ✅ **Engagement accru** : Les utilisateurs gardent l'app sur leur écran d'accueil
- ✅ **Résilience** : L'app continue de fonctionner même en cas de panne réseau

---

## 🏗️ Architecture Technique

### Stack
- **Service Worker** : Gestion du cache et mode hors-ligne
- **Manifest.json** : Configuration PWA (icônes, nom, couleurs, orientation)
- **Next.js** : Support PWA natif avec `next-pwa`
- **Workbox** : Stratégies de cache avancées (Network First, Cache First, Stale While Revalidate)

### Stratégies de cache

```
1. Pages statiques (/, /landing, /sign-in, /sign-up)
   → Cache First (affichage instantané)

2. API Routes (/api/*)
   → Network First avec fallback cache (données fraîches prioritaires)

3. Assets statiques (images, CSS, JS)
   → Cache First avec versioning (performance maximale)

4. Données temps réel (tickets, WebSocket)
   → Network Only (pas de cache pour éviter les données obsolètes)
```

          maxAgeSeconds: 24 * 60 * 60

      }

    {

      handler: 'NetworkFirst',

    {
      urlPattern: ({ url }) => {
        const isSameOrigin = self.origin === url.origin
        if (!isSameOrigin) return false
        const pathname = url.pathname

### ✅ Étape 5 : Gérer le mode hors-ligne dans l'UI
- [ ] **5.1** Créer un composant `OfflineIndicator` :
  ```tsx
  'use client'
  import { useEffect, useState } from 'react'

  export default function OfflineIndicator() {
    const [isOnline, setIsOnline] = useState(true)

    useEffect(() => {
      setIsOnline(navigator.onLine)

      const handleOnline = () => setIsOnline(true)
      const handleOffline = () => setIsOnline(false)

      window.addEventListener('online', handleOnline)
      window.addEventListener('offline', handleOffline)

      return () => {
        window.removeEventListener('online', handleOnline)
        window.removeEventListener('offline', handleOffline)
      }
    }, [])

    if (isOnline) return null

    return (
      <div className="fixed top-0 left-0 right-0 bg-warning text-warning-content p-2 text-center z-50">
        ⚠️ Mode hors-ligne - Certaines fonctionnalités peuvent être limitées
      </div>
    )
  }
  ```

### ✅ Étape 6 : Ajouter un bouton d'installation PWA
- [ ] **6.1** Créer un composant `InstallPWA` :
  ```tsx
  'use client'
  import { useEffect, useState } from 'react'

  export default function InstallPWA() {
    const [deferredPrompt, setDeferredPrompt] = useState<any>(null)
    const [showInstallButton, setShowInstallButton] = useState(false)

    useEffect(() => {
      const handler = (e: Event) => {
        e.preventDefault()
        setDeferredPrompt(e)
        setShowInstallButton(true)
      }

      window.addEventListener('beforeinstallprompt', handler)

      return () => window.removeEventListener('beforeinstallprompt', handler)
    }, [])

    const handleInstall = async () => {
      if (!deferredPrompt) return

      deferredPrompt.prompt()
      const { outcome } = await deferredPrompt.userChoice

      if (outcome === 'accepted') {
        setShowInstallButton(false)
      }

      setDeferredPrompt(null)
    }

    if (!showInstallButton) return null

    return (
      <button onClick={handleInstall} className="btn btn-primary btn-sm">
        📱 Installer l'app
      </button>
    )
  }
  ```

### ✅ Étape 8 : Gérer les mises à jour du Service Worker
- [ ] **8.1** Créer un composant `UpdateNotification` :
  ```tsx
  'use client'
  import { useEffect, useState } from 'react'

  export default function UpdateNotification() {
    const [showUpdate, setShowUpdate] = useState(false)

    useEffect(() => {
      if ('serviceWorker' in navigator) {
        navigator.serviceWorker.ready.then((registration) => {
          registration.addEventListener('updatefound', () => {
            const newWorker = registration.installing
            if (newWorker) {
              newWorker.addEventListener('statechange', () => {
                if (newWorker.state === 'installed' && navigator.serviceWorker.controller) {
                  setShowUpdate(true)
                }
              })
            }
          })
        })
      }
    }, [])

    const handleUpdate = () => {
      window.location.reload()
    }

    if (!showUpdate) return null

    return (
      <div className="toast toast-top toast-center">
        <div className="alert alert-info">
          <span>🔄 Une nouvelle version est disponible !</span>
          <button onClick={handleUpdate} className="btn btn-sm">
            Mettre à jour
          </button>
        </div>
      </div>
    )
  }
  ```

## 🔐 Sécurité & Bonnes pratiques

### HTTPS obligatoire
- ✅ **Prérequis** : Les PWA nécessitent HTTPS (sauf localhost)
- ✅ **Production** : Vercel/Railway fournissent HTTPS par défaut

### Gestion du cache
- ✅ **Versioning** : Le Service Worker se met à jour automatiquement
- ✅ **Nettoyage** : Les anciens caches sont supprimés automatiquement
- ✅ **Taille** : Limiter le cache à 50 Mo max (quotas navigateur)

### Données sensibles
- ✅ **Pas de cache** : Ne jamais cacher les tokens d'authentification
- ✅ **Network Only** : Routes API sensibles en Network Only
- ✅ **Expiration** : Définir des durées de cache appropriées

## 📝 Notes & Remarques

### Limitations connues
- **iOS Safari** : Pas de support du bouton d'installation automatique (manuel via "Partager")
- **iOS Safari** : Service Worker limité (pas de background sync)
- **Stockage** : Quotas variables selon le navigateur (50 Mo à 1 Go)

### Optimisations futures
- **Background Sync** : Synchroniser les actions hors-ligne quand la connexion revient
- **Push Notifications** : Notifications natives (nécessite HTTPS + permission)
- **App Shortcuts** : Raccourcis rapides dans le menu contextuel de l'icône
- **Share Target** : Permettre de partager du contenu vers SmartQueue

### Alternatives
- **Capacitor** : Pour une vraie app native (iOS/Android) si nécessaire
- **Electron** : Pour une app desktop (Windows/Mac/Linux)
